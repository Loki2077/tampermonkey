# 抖音增强脚本

## 功能特性
✅ 动态清晰度切换（1080P/720P/480P）  
✅ 直播视频自动跳过  
✅ 快捷键控制面板（J键切换）  
✅ 智能DOM元素检测  
✅ 样式隔离系统
✅ P键沉浸模式（全屏并清除不必要元素）
✅ J键清屏功能

## 配置参数
```javascript
// ==配置面板==
const config = {
  // 白名单域名
  whitelist: ['www.douyin.com'],
  // 自动选择最高清晰度
  autoQuality: true,
  // 跳过直播视频
  skipLive: true,
  // 全屏模式
  autoFullscreen: true,
  // 调试模式
  debugMode: false
};
```

## 快捷键说明
| 快捷键 | 功能 | 触发效果 |
|-------|------|---------|
| P | 沉浸模式 | 1. 全屏播放\n2. 移除侧边栏\n3. 隐藏评论区\n4. 清除浮层广告 |
| J | 清屏模式 | 1. 隐藏弹幕\n2. 移除礼物动画\n3. 隐藏悬浮按钮\n4. 保留核心播放器 |
