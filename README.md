# 油猴脚本集合

## 项目概述
这是一个油猴脚本集合项目，包含多个实用的浏览器脚本，用于增强网站功能。

## 文档导航
- [抖音增强脚本说明](./DOUYIN_README.md)
- [DeepSeek辅助脚本说明](./DEEPSEEK_README.md)

## 安装方法

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击对应脚本的安装链接，或手动将脚本代码复制到Tampermonkey中创建新脚本
3. 刷新目标网站页面即可使用
